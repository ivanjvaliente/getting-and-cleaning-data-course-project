# Getting and Cleaning Data Course Project : « run_analysis.R » script
This script process the initial data in order to get tidy data set that can be used for later analysis.
The script follows step by step the instructions provided for the project.
The "codebook.Rmd"file provides more details about the data set as well as the main steps of the data processing.
The comments provided in the « run_analysis.R » script, provides also detailed information about each operation applied to the data, displaying the partial results to show how the code works.
